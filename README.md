> **AI-Powered Career Guidance Platform for Indian Rural Youth**

CareerAI is an intelligent web application that helps users discover their perfect career path using Machine Learning and Artificial Intelligence. It provides personalized career recommendations, college suggestions, job listings, and government scheme information.

## ✨ Features

### 🎯 Core Features
- **AI-Powered Career Recommendations** - ML-based career matching based on skills, education, and location
- **Google Gemini AI Integration** - Advanced career suggestions and profile analysis
- **Interactive Dashboard** - User-friendly dashboard with real-time search
- **Multi-Language Support** - English and Hindi with Voice Assistant
- **Voice Assistant** - Text-to-Speech support for accessibility

### 🏫 College & Education
- **College Recommendations** - Top Indian and International colleges for each career
- **Career Pathway** - Step-by-step career roadmap
- **Education Level Filtering** - Recommendations based on education level

### 💼 Jobs & Career
- **Live Job Search** - Real-time job listings from multiple APIs
- **Job Saving** - Save and track interesting jobs
- **Application Tracking** - Track job applications status

### 🏛️ Government Schemes
- **Scheme Recommendations** - Relevant government schemes for each career
- **Eligibility Criteria** - Detailed scheme information
- **Application Links** - Direct links to apply

### 👤 User Features
- **User Authentication** - Secure login and registration
- **Profile Management** - Update user profile and preferences
- **Saved Careers** - Save and manage favorite careers
- **Search History** - Track past career searches
- **Career Ratings** - Rate and provide feedback on careers
